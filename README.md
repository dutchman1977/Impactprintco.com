# Impact Print Co. Landing Page

Production-ready React + Tailwind + Framer Motion landing page for **Impact Print Co.**

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `index.html` – Vite entry HTML
- `src/main.jsx` – React bootstrap
- `src/App.jsx` – app shell
- `src/ImpactPrintHomepage.jsx` – full landing page component
- `src/styles.css` – Tailwind layers + typography setup
- `tailwind.config.js` – Tailwind content paths
- `postcss.config.js` – Tailwind + Autoprefixer pipeline
- `vite.config.js` – Vite React configuration
